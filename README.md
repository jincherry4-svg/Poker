# 視窗程式設計 (II) - 五張撲克牌遊戲與下注功能

這是一個基於 C# Windows Forms 開發的五張撲克牌遊戲。玩家可以透過押注金額、發牌、換牌以及判斷牌型來進行遊戲。本專案包含完整的牌型賠率計算與資金管理系統。


## 遊戲功能
* **資金管理**：初始資金為 1,000,000，支援自定義押注金額。
* **破產重生**：當剩餘資金不足以支付押注額時，系統會提示並可重設資金為 1,000,000。
* **洗牌/發牌系統**：使用隨機演算法確保公平性，發牌後可選擇要更換的牌。
* **牌型判斷**：自動判斷玩家最終手牌的牌型，並根據官方賠率表計算獎金。

## 如何執行
1. 下載本專案原始碼。
2. 使用 **Visual Studio 2022** 開啟 `Poker.sln`。
3. 按下 `F5` 或點擊「開始」進行編譯與執行。
4. **操作順序**：輸入押注金額 -> 點擊「押注」 -> 點擊「發牌」 -> 點擊牌面選擇換牌 -> 點擊「換牌」 -> 點擊「判斷牌型」。

* **框架**：.NET Framework 4.7.2 / .NET 6.0 (依實際情況填寫)
